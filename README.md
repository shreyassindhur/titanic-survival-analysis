# Titanic Survival Analysis

This repository contains a Jupyter Notebook for analyzing the Titanic dataset and predicting passenger survival. The notebook explores data cleaning, visualization, and machine learning techniques to build predictive models.

## Contents
- `Titanic_Survival_(8)_(2) (3) (1).ipynb`: Main notebook for Titanic survival analysis

## Description
The notebook walks through:
- Loading and exploring the Titanic dataset
- Data preprocessing and feature engineering
- Visualizing key patterns and relationships
- Building and evaluating machine learning models to predict survival

This project is intended for educational purposes and demonstrates a typical data science workflow using Python and Jupyter Notebooks.
